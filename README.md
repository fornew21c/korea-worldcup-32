# 🇰🇷 대한민국 32강 경우의 수 시뮬레이터

대한민국은 현재 **1승 2패 · 승점 3 · 골득실 −1 · 다득점 2**로, 각 조 3위 12개 팀 중 **상위 8위 이내(와일드카드)**에 들면 32강에 진출합니다. 아직 D~L 9개 조의 최종전이 안 치러져 순위는 미정입니다.
남은 D~L조 최종전 결과에 따라 순위가 변동될 수 있으며, 한국에 유리한 9개 조 시나리오 중 **3개만 충족하면 진출 확정**으로 가정해 시뮬레이션합니다.
각 조 카드를 눌러 결과를 켜고 끄면 상단 전광판이 실시간으로 진출/탈락을 판정합니다.

- 🎲 랜덤 시뮬레이션 / 초기화
- 진출선(3) 게이지 + 충족 카운터
- 3개 도달 시 컨페티 🎉
- 모바일 · 키보드 대응, `prefers-reduced-motion` 존중

단일 HTML 파일(`index.html`)로 동작하는 정적 페이지입니다. 외부 의존성 없음.

> 공유 인포그래픽 기반 참고용 시나리오이며, 실제 대진·순위 규정과 다를 수 있습니다.
